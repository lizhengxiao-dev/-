# -金融韧性综合指数计算工具 (Financial Resilience Index Calculator)本项目是一个基于 Python 的自动化数据处理工具，旨在计算中国各省份在 2011-2022 年间的金融韧性综合指数。程序集成了数据清洗、标准化、熵值法权重计算及综合评分生成等核心功能。1. 项目核心流程项目严格遵循以下五步科学计算流程：数据准备：读取包含多年份数据的 Total.xlsx，每一年对应一个工作表（Sheet）。数据清洗：自动识别指标列，处理表头杂质（空格、特殊字符），并对缺失值进行均值填充。指标标准化：正向指标：采用最大值标准化。负向指标：采用“倒数转换 + 最大值标准化”处理，确保指标方向一致。熵值法权重计算：利用信息熵原理，逐年独立计算 13 个核心指标的权重，反映不同年份的指标贡献差异。得分合成：通过加权求和公式计算各省份每年的金融韧性综合指数（范围 [0, 1]）。2. 指标体系说明程序共处理 13 个核心指标，分为两类：类型指标名称正向指标 (11个)保险深度、金融机构存款余额、金融机构贷款余额、社会融资规模增量、上市公司数量、数字普惠金融指数、存贷比、人均GDP、第三产业占比、金融科技专利授权量、耦合协调度负向指标 (2个)不良贷款率、外贸依存度3. 环境准备确保你的电脑已安装 Python 3.8+ 及以下库：Bashpip install pandas numpy openpyxl
4. 文件结构main.py：核心 Python 执行脚本。Total.xlsx：原始数据文件（需包含 2011-2022 各年份命名的 Sheet）。金融韧性综合指数_最终版.csv：程序运行后自动生成的面板数据结果。5. 快速上手将原始数据重命名为 Total.xlsx。将 main.py 与 Total.xlsx 放置在同一目录下。在终端或命令行运行：Bashpython main.py
运行完成后，在同目录下查看生成的 .csv 结果文件。6. 数学逻辑详解标准化处理正向指标: $x'_{ij} = \frac{x_{ij}}{\max(x_{j})}$负向指标: $x'_{ij} = \frac{1/x_{ij}}{\max(1/x_{j})}$熵值法权重 (Entropy Method)程序会自动执行以下计算：计算各指标的比重 $p_{ij} = \frac{x'_{ij}}{\sum x'_{ij}}$。计算第 $j$ 项指标的熵值 $e_j = -k \sum (p_{ij} \ln p_{ij})$。计算差异系数 $g_j = 1 - e_j$。确定最终权重 $w_j = \frac{g_j}{\sum g_j}$。7. 注意事项表头要求：Excel 工作表的第 2 行应为指标名称，第 3 行开始为数据。省份一致性：程序会自动剔除省份列中的空行或无效字符（如 'nan'）。除零处理：程序内置了极小值（1e-9）平滑处理，防止在对数计算或倒数计算时报错。
