# 金融韧性综合指数计算工具 (Financial Resilience Index Calculator)

本项目是一个基于 Python 的自动化数据处理工具，旨在计算中国各省份在 2011-2022 年间的**金融韧性综合指数**。程序集成了数据清洗、标准化、熵值法权重计算及综合评分生成等核心功能。

## 1. 项目核心流程

项目严格遵循以下五步科学计算流程：

1.  **数据准备**：读取包含多年份数据的 `Total.xlsx`，每一年对应一个工作表（Sheet）。
2.  **数据清洗**：自动识别指标列，处理表头杂质（空格、特殊字符），并对缺失值进行均值填充。
3.  **指标标准化**：
    * **正向指标**：采用最大值标准化。
    * **负向指标**：采用“倒数转换 + 最大值标准化”处理，确保指标方向一致。
4.  **熵值法权重计算**：利用信息熵原理，逐年独立计算 13 个核心指标的权重，反映不同年份的指标贡献差异。
5.  **得分合成**：通过加权求和公式计算各省份每年的金融韧性综合指数（范围 [0, 1]）。

## 2. 指标体系说明

程序共处理 13 个核心指标，分为两类：

| 类型 | 指标名称 |
| :--- | :--- |
| **正向指标** (11个) | 保险深度、金融机构存款余额、金融机构贷款余额、社会融资规模增量、上市公司数量、数字普惠金融指数、存贷比、人均GDP、第三产业占比、金融科技专利授权量、耦合协调度 |
| **负向指标** (2个) | 不良贷款率、外贸依存度 |
